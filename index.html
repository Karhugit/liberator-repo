First Install: Liberator (Kodi Add-on)

1
In Kodi, go to Settings → File Manager → Add Source.
2
Enter the following URL as the source, then give it a name (e.g. Liberator):
https://karhugit.github.io/liberator-repo/zips/
Click the URL above to copy it
3
Go to Settings → Add-ons → Install from zip file, select the Liberator source, then install repository.liberator-1.0.0.zip.
4
Go to Install from repository → Liberator Repository and install Liberator.
ℹ️ The File Manager source URL ends in /zips/ — this is where Kodi reads addons.xml to discover available add-ons. Do not use the zip file URL directly as the source.
